## Getting Started

- Clone
- Install Xcode 10
- Get the API Key from https://developers.themoviedb.org/3/
- Put API Key in MovieService.swift apiKey constant or you can use mine
- pod install
- Click Suggestion for YoutubePlayer code update on build
